# 习题二

## 要求

读取一行整数，输出所有重复的整数（相同整数输出一次即可），并按这些整数在输入中首次重复（即第二次出现）的顺序排列。

时间复杂度为`O(n)`

## IO格式

输入：

一行整数，用单个空格分隔

输出：

一行整数，包含且仅包含所有重复整数，用单个空格分隔

## 示例

> 输入：
> 
>   ```
>   4 2 3 7 8 3 1 2 3
>   ```
>
> 输出：
> 
>   ```
>   3 2
>   ```
> 
> 解释：
> 
> `3`在索引5处首次重复，`2`在索引7处首次重复

## 提示

可以结合`HashMap`和`ArrayList`

## 进阶思考

（不强制要求）
